Combining curves

The power of PATH command

You can concatenate the curve sections into one single path command
path d="M 10 85 C 50 160, 100 160, 150 85 C 190 10, 230 10, 270 85"fill="transparent" stroke="black" stroke-weight="4" />

Using the S command (String together) will continue the curve, use x1 y1 xy (where xy are the end points).
